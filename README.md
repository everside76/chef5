# 저녁 준비 모바일 플래너

정적 HTML로 만든 모바일 저녁 준비 플래너입니다. 레시피, 장보기 체크, 조리 순서를 한 화면에서 확인할 수 있고 PWA로 홈 화면 설치를 지원합니다.

레시피와 장보기 항목은 브라우저 IndexedDB에 저장됩니다. 앱 안에서 레시피와 재료를 추가, 수정, 삭제할 수 있으며 체크 상태는 로컬에 유지됩니다.

## 로컬 실행

```bash
python -m http.server 4173 --bind 127.0.0.1
```

브라우저에서 `http://127.0.0.1:4173/recipe_mobile_web.html`을 엽니다.

## 배포

GitHub Pages에서 루트 디렉터리를 배포하도록 구성하면 외부에서 접근할 수 있습니다.
